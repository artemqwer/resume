# Web Resume - Artem Lytvyn

A modern, interactive web resume built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Responsive Design**: Looks great on mobile and desktop.
- **Animations**: Smooth transitions using Framer Motion.
- **Project Showcase**: Interactive cards for your projects.
- **Skills Section**: Categorized technical skills.
- **Contact Form**: Easy access to social links.

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd web-resume
    ```

2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) (or the port shown in terminal) with your browser.

## Customization

-   **Hero Section**: Edit `components/Hero.tsx` to update your bio or profile picture.
-   **Skills**: Edit `components/Skills.tsx` to add or remove skills.
-   **Projects**: Edit `components/Projects.tsx` to update your project list.
-   **Contact**: Edit `components/Contact.tsx` to update your links.
